Quantity Changer On CheckOut For WooCommerce is an add-on for WooCommerce which shows quantity box and remove option in the checkout page.

== Description ==
Quantity Changer On CheckOut For WooCommerce is a simple and light weight add-on for WooCommerce which simply allows your users to update the quantity of products by showing quantity box and gives the options to remove the products directly from the checkout page.

This option gives ability to your users to modify their cart content directly from the checkout page without going to the cart page.

== Installation ==
1. Upload "Quantity Changer On CheckOut For WC" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Installation complete
